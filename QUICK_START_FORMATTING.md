# Quick Start: Rich Text Formatting

## For Admin Users - How to Make Text Bold

### Step 1: Go to Case Studies Management
- Open Admin Dashboard
- Click "Manage Case Studies" (or Blogs/Articles)

### Step 2: Click Edit or Upload
- For new content: Click "Upload Case Study"
- For existing: Click "Edit" on any case study card

### Step 3: Access the Rich Text Editor
In the "Content" section, you'll see a text editor with formatting buttons

### Step 4: Make Text Bold
1. **Select the text** you want to make bold
2. **Click the "B" button** (Bold) in the toolbar
   - Or use keyboard shortcut: **Ctrl+B** (Windows) / **Cmd+B** (Mac)

### Step 5: Add Headings
1. Select your text
2. Click the **dropdown menu** at the far left
3. Choose **H1, H2, or H3** for different heading sizes
   - H1 = Largest heading
   - H2 = Medium heading
   - H3 = Smallest heading

### Step 6: Other Formatting Options

| Want to do... | Action |
|---|---|
| Make text italic | Click **I** button or Ctrl+I |
| Underline text | Click **U** button or Ctrl+U |
| Strikethrough | Click ≈ button |
| Create bullet list | Click **•** button |
| Create numbered list | Click **☰** button |
| Add a quote | Click **"** button |
| Change text color | Click **A** button with color palette |
| Add a link | Click **🔗** button and paste URL |
| Insert image | Click **🖼️** button |
| Clear formatting | Click **✕** button |

### Step 7: Save Your Work
- Review your formatted content
- Click "Confirm" to save
- Your formatted content will appear on the website!

## Example

**Before (plain text):**
```
Product Overview
This is our main product. It is high quality. You will love it.
Features
- Feature 1
- Feature 2
```

**After (with formatting):**
```
Product Overview (H2 heading)
This is our main product. It is high quality. You will love it.
Features (Bold text)
- Feature 1
- Feature 2
```

## Pro Tips

✨ **Make Headings Stand Out**: Use H1 for main title, H2 for sections, H3 for subsections

🎨 **Use Colors Wisely**: Green for important, Red for warnings, Blue for links

📝 **Lists for Clarity**: Use bullet points for features or bullet lists for related items

🔗 **Add Links**: Connect to other pages or external resources

💾 **No Auto-Save**: Always click "Confirm" when done!
